# Beijing Claude Translator

一个很小的 Windows 悬浮工具，主要给 Claude 用户用。

![主界面截图](assets/screenshot-main.png)

## 功能

- 悬浮显示北京时间。
- 中文转英文，方便把中文想法发给 Claude。
- 英文转中文，方便看英文回复或资料。
- 关闭窗口后可留在系统托盘。
- 有安装引导，可以直接使用，也可以安装到本机。
- 支持开机自启。
- 支持日间 / 夜间模式。
- 支持从 GitHub 检查更新。

## 适合谁

如果你把电脑时区改成海外，但仍然想看北京时间，并且想用中文输入、英文和 Claude 聊天，这个工具就是为这个场景做的。

## Windows 使用方式

下载：

```text
BeijingClaudeTranslator.exe
```

双击就能打开，不需要解压，不需要安装。

第一次打开会出现安装引导：

- 想简单试用，点“直接使用”。
- 想长期使用，点“安装并打开”。

## 说明

默认使用公共在线翻译接口，适合日常轻量使用。公共接口慢或失败时，可以稍后重试。

长期稳定使用可以自己配置 Microsoft Translator 或 DeepL。
